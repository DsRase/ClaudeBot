# Структура проекта
```bash
├── resources # ресурсы: png, mp3 и т. д.
├── src # исходный код
│   ├── agent # агент по работе с LLM (langchain)
│   ├── bot # бот для ТГ
│   ├── config # все настройки
│   ├── utils # доп. функционал: логгер, хелперы и т. д.
│   └── tools # тулы для агента: работа с гит, памятью и т. д.
└── tests # тесты
    └── # полностью повторяю src, только для тестов
```

# Тестирование
*Все тесты находятся в директории `tests`. Директория должна быть полностью идентична `src` и 
хранить в себе тесты для каждого отдельного модуля*
